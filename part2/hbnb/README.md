# HBnB Project

## Structure du Projet
- `app/` : Contient le code de l'application Flask.
- `api/` : Gère les routes API.
- `models/` : Contient les modèles métiers.
- `services/` : Gère la logique métier via le Facade Pattern.
- `persistence/` : Fournit un stockage en mémoire.

## Installation
```bash
pip install -r requirements.txt
python run.py
